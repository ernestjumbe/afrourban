# Guide

Compact agent-facing reference set for AfroUrban Design 4.

## Files

- `FRONTEND_DESIGN_SYSTEM.md`: global rules, tokens, typography, motion, layout vocabulary.
- `COMPONENT_GUIDE.md`: how individual components work and when to use them.
- `RECIPES.md`: common page compositions and assembly rules.
- `examples/README.md`: canonical implementation references inside this repo.

## Usage Order

1. Read `FRONTEND_DESIGN_SYSTEM.md` for overall constraints.
2. Read `COMPONENT_GUIDE.md` for component-level rules.
3. Read `RECIPES.md` for page assembly.
4. Open matching files from `examples/README.md` before implementing.

## Defaults

- Design family: Design 4
- Theme default: dark
- Primary accent: gold
- Secondary accent: terracotta
- Major headings: serif
- UI text: sans
